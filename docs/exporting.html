Export & Save

Export your work to 5 formats, or save the full scene as a native JSON file to reload later.

Export Formats

All export buttons are in the top toolbar. Click any to download the file immediately.

DXF

Export DXF

AC1015 (R2000) format using LWPOLYLINE entities. Units are in meters. Best for sending to CAD software for further documentation or detailing.

Compatible with: Rhino, AutoCAD, BricsCAD, LibreCAD

OBJ

Export OBJ

Wavefront OBJ with vertex normals and named groups per object. Good all-purpose 3D mesh format for rendering and 3D printing.

Compatible with: Blender, Cinema 4D, 3ds Max, Maya, SketchFab, most renderers

3DS

Export 3DS

Binary 3D Studio format with proper MAIN3DS/EDIT3DS chunk structure. Legacy format with wide compatibility across older tools.

Compatible with: 3ds Max, many CAD and visualization tools

PDF

Export PDF / PNG

Captures the current viewport as-is — what you see is what you get. Set up the exact view you want first (elevation, perspective, plan view) then export. Great for quick presentations and sharing.

Use with: Plan view for floor plans, Front/Right/Left/Back for elevations, 3D for perspective presentations

Tip for clean elevations: Switch to Front (5), Right (6), or another elevation view, turn on Wireframe (Z) if needed, then export PDF for a clean 2D drawing.

Save & Open (Native Format)

The Save and Save As... buttons in the top toolbar save your entire scene as a .json file. This is AI-Architek's native format and saves everything:

✓ All objects and geometry
✓ Object colors and groups
✓ Lighting settings
✓ Sky and environment
✓ All saved camera views
✓ Camera frustum cones
✓ Solar position settings
✓ Grid and snap settings

To reload a saved scene, click Open in the top toolbar and select your .json file. Everything restores exactly as you left it.

Good habit: Use Save As... with version numbers to keep backups — e.g. project-v1.json, project-v2.json. This way you can always roll back to an earlier version.
