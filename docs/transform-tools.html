Transform Tools

Edit, move, duplicate, and reshape objects. All transforms support multi-select and full undo.

Move & Copy

M Move

3-step operation: 1) Select the object(s) → 2) Click a base point (the handle to grab) → 3) Click the destination. Works on multi-select — all selected objects move together as a group.

K Copy

Identical 3-step flow as Move, but places a duplicate at the destination and leaves the original untouched. Multi-select copies all selected objects.

Ctrl+J Array

Creates a 3D grid of copies at X × Y × Z count. Set the spacing and count in the dialog that appears.

Orientation

X Mirror

Reflects selected objects across a line you draw. Click two points to define the mirror axis. Works in 2 axes.

Rotate

Click to set the pivot point, then type an angle in degrees. Use the X / Y / Z axis selector to choose which axis to rotate around.

Scale

Scale selected objects along X, Y, Z, or uniformly. Type a scale factor or drag interactively.

Geometry Editing

Stretch / Extrude Face

Click any face on a solid to select it, then drag or type a distance to push or pull it. This is the primary way to resize individual faces after drawing.

Chamfer V6

Click a vertical edge on any solid, then type a bevel distance. Cuts the edge at an angle to create a flat chamfer. Fully undoable.

Fillet V6

Click a vertical edge and type an arc radius. Rounds the edge with a smooth curve. Each corner can be filleted independently — apply it to all 4 corners of a box one by one for rounded walls.

Delete Face

Removes a single face from a solid without deleting the whole object. Useful for opening up a wall or removing a surface.

Boolean Operations

Union

Merges two overlapping solids into one combined shape. Select both objects, then click Union.

Subtract

Cuts one shape out of another. Select the base solid first, then the cutter shape, then click Subtract. Great for punching door or window openings into walls.

Wall Tools

Trim

Cuts a wall at its intersection with another wall. Click the wall to trim, then click the boundary wall.

Extend

Extends a wall until it meets a boundary line or another wall.

Groups & Selection

Group / Ungroup

Group stamps a shared ID on selected objects so they move and transform together. Changing the color of one group member changes all of them. Ungroup releases them.

Ctrl+A Select All

Selects every object in the scene. Combine with Move, Copy, Delete, or any transform.

Undo: Ctrl+Z undoes one step. Ctrl+Y redoes. Multi-object moves all reverse together in one undo step.
